Detailed Description

sec_checker

struct mac_ax_pkt_data - packet information of data type : Length of header+LLC. For example,

  1. 802.11 MPDU without encryption HEADERwLLC_LEN = (MAC header (without IV ) + LLC ) / 2
  2. 802.11 MPDU encryption without HW_AES_IV HEADERwLLC_LEN = (MAC header (without IV ) + LLC ) / 2 3.802.11MPDU encryption with HW_AES_IV HEADERwLLC_LEN = (MAC header(reserved IV length)+LLC)/2 4.ETHERNET II MSDU without encryption HEADERwLLC_LEN = (DA+SA+TYPE) /2 5.ETHERNET II MSDU encryption without HW_AES_IV HEADERwLLC_LEN = (DA+SA+TYPE) /2 6.ETHERNET II MSDU encryption with HW_AES_IV HEADERwLLC_LEN = (DA+SA+TYPE) /2 7.SNAP MSDU without encryption HEADERwLLC_LEN = (DA+SA+LEN+LLC) /2 8.SNAP MSDU encryption without HW_AES_IV HEADERwLLC_LEN = (DA+SA+LEN+LLC) /2 9.SNAP MSDU encryption with HW_AES_IV HEADERwLLC_LEN = (DA+SA+LEN+LLC) /2 : Channel index, MAC_AX_CH_DMA_CH0~MAC_AX_CH_DMA_CH11. : MAC ID.
